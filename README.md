# bank-marketing-campaign
Dalam proyek ini, kita menggunakan dataset Bank Marketing Campaign yang umum digunakan dalam supervised machine learning. Dataset ini berisi data demografis dan perilaku klien dalam kampanye pemasaran bank yang bertujuan memprediksi apakah seorang nasabah akan menyetujui produk term deposit setelah contact kampanye telemarketing.
Proses yang dilakukan meliputi:
Eksplorasi Data Awal (EDA) – Memahami struktur data, tipe variabel, distribusi kelas target, serta hubungan antara fitur dan target.
Preprocessing & Pembersihan Data – Melakukan encoding variabel kategori, penanganan missing values, dan penyiapan dataset agar kompatibel dengan model ML.
Pembagian Dataset – Data dibagi menjadi training dan testing untuk evaluasi model yang objektif.
Pemodelan Machine Learning – Menerapkan model Decision Tree serta teknik ensemble (contoh: Bagging, Random Forest) untuk mempelajari pola dari data dan menghasilkan prediksi.
Evaluasi Model – Menggunakan metrik seperti Accuracy, Recall, Precision, dan F1-Score untuk menilai performa prediksi model pada data uji.
